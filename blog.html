What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a hierarchical tree of objects that can be manipulated with programming languages like JavaScript. The DOM allows developers to access, modify, and interact with HTML or XML documents dynamically.

What is event DeleGation in the Context of the DOM, and wht is it Useful?

Event Delegation is a powerful technique in JavaScript used to handle events on elements that may not exist at the time the event listener is attached. Instead of adding an event listener to each individual element, you can attach a single event listener to a common ancestor (parent) element. This parent element captures events that occur on its child elements, allowing you to manage events more efficiently.
